What is Forgetful Fairy?

Forgetful Fairy was motivated by the fact that iOS only allows the user to place two reminders for calendar events, with no variable snooze functionality. Forgetful Fairy reminds you of important events via text-message. The text-messages arrive in increasing frequency as your event approaches. You will receive a total of 8 reminders as long as your event is at least two months in the future. Forgetful Fairy utilizes the Twilio API and is built with Node.js. Lastly, Forgetful Fairy is optimized for mobile devices. The source code for ForgetfulFairy may be found on Github.
